## Rodando aplicação localmente

1. Clonar o repositório:

   ```bash
    git clone https://github.com/progmateus/knights-frontend.git
   ```

2. Navegar até o diretório do projeto::

   ```bash
    cd knights-frontend
   ```

3. Instalar dependências

   ```bash
    npm i
   ```

4. Rodar aplicação
   ```bash
    npm run dev
   ```


## Tecnologias utilizadas
- Vue
- Tailwind
- Dayjs
- Tanstack Query
- Vue Router
- Vite
